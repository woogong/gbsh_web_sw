DOM 공부

웹 페이지가 로딩되면 브라우저는 Document Object Model을 생성합니다.
HTML DOM은 Object들의 tree 구조입니다.
Object model을 이용하여, 동적(dynamic) HTML에 필요한 모든 능력을 부여받을 수 있습니다.
  • HTML element 들을 변경할 수 있습니다.
  • HTML 속성들을 변경할 수 있습니다.
  • CSS style을 변경할 수 있습니다.
  • HTML element을 제거할 수 있습니다.
  • 새 HTML element을 추가할 수 있습니다.
  • HTML element의 이벤트를 재정의 할 수 있습니다.
  • HTML element에 새 이벤트를 만들 수 있습니다.

What is the DOM

The DOM is a W3C (World Wide Web Consortium) standard.
The DOM defines a standard for accessing documents:
The HTML DOM is a standard for how to get, change, add, or delete HTML elements.
